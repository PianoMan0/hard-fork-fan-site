Hard Fork

Hard Fork is a New York Times tech podcast hosted by Kevin Roose and Casey Newton. Almost 200 episodes have been published to date. The podcast is weekly and explores the latest trends in tech.

The Hosts

Kevin Roose is a tech columnist for The New York Times.

Casey Newton is a mysterious guy.


The Forkiverse

The Hard Fork hosts have teamed up with PG Vogt, host of the podcast Search Engine, to create the Forkiverse. The Forkiverse is a social media site that is a part of the Fediverse. In it's first week it gathered thousands lf users and it is still growing.
